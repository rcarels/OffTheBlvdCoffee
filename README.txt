OFF THE BLVD COFFEE WEBSITE

How to preview:
1. Open this folder in Visual Studio Code.
2. Open index.html.
3. Right-click and choose Open with Live Server.

Functional features included:
- Mobile-responsive layout.
- Enlarged header logo without changing nav text sizing.
- Fully fillable quote form with validation.
- Quote form opens the visitor's email app using a placeholder email address: hello@offtheblvdcoffee.com.
- Quote form saves the latest submission in the visitor's browser localStorage.
- Event calendar section with search and event-type filter.
- Downloadable .ics calendar file for scheduled events.
- Instagram link to https://www.instagram.com/offtheblvdcoffee/
- Placeholders for missing pricing, events, exact addresses, and extra gallery photos.

Important next edits:
